### 关键教训
- GitHub Actions 在海外运行，可以直接用 raw.githubusercontent.com，不需要 jsdelivr 镜像
- `npm ci` 要求 package-lock.json 存在，CI 应做兼容处理
- IPTV 源大多不支持 HEAD 请求，测试应用 GET + Range
- rtp/rtsp 组播源无法通过 HTTP 测试，应跳过
