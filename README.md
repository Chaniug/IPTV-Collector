# IPTV 电视直播源

收集整理国内可用的电视频道直播源，支持直接订阅使用。

## 📡 使用方法

### 方式一：订阅链接（推荐）

复制以下链接到 IPTV 播放器中订阅：
```
https://chaniug.github.io/IPTV-Collector/iptv.m3u
```

### 方式二：下载 M3U 文件

直接下载 `iptv.m3u` 文件，导入播放器即可。

## 📱 推荐播放器

- **TVBox** - Android 平台，功能强大
- **PotPlayer** - Windows 桌面端
- **VLC** - 跨平台开源播放器
- **IINA** - macOS 专用

## 📺 频道分类

- **央视** - CCTV 1-17 全系列
- **卫视** - 湖南卫视、浙江卫视、江苏卫视、东方卫视、凤凰卫视等
- **地方台** - 全国各省市地方卫视
